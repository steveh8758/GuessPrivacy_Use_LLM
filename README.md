### LLM
1. 打開 `Promt.txt`。
2. 將內容複製到語言模型中輸入。
3. 將匯出的 `[LINE] 與xxx的x聊天_ooo.json` 檔案以文字文件打開，並且複製進模型中。
4. 複製完後，開始問問題。
